# Intelligent Driving Music Agent 

## Project Overview
An intelligent recommendation agent that acts as a "Virtual DJ" for drivers, dynamically selecting songs based on historical listening patterns, genre, and artist preferences.

## Getting Started
1. **Clone the repo:** `git clone <repo-url>`
2. **Create a virtual environment:** `python -m venv venv`
3. **Activate venv:** - Windows: `.\venv\Scripts\activate`
   - Mac/Linux: `source venv/bin/activate`
4. **Install dependencies:** `pip install -r requirements.txt`

## Directory Structure
* `src/`: Core logic and agent scripts.
* `notebooks/`: Data exploration and model testing.
* `data/`: Local datasets (ignored by git).
* `models/`: Intelligent Agent Tool
